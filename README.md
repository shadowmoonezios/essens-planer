# Essens Planer

Willkommen zum Essens Planer! Diese Anwendung hilft dir, deine Mahlzeiten für die Woche zu planen und erleichtert das Einkaufen von Zutaten.

## Features
- Erstellung und Verwaltung von Mahlzeitenplänen
- Speicherung und Suche von Rezepten
- Automatische Generierung von Einkaufslisten basierend auf deinem Mahlzeitenplan
- Benutzerfreundliches Interface für einfache Navigation und Bedienung

## Technologien
- Frontend: React.js
- Backend: Node.js mit Express
- Datenbank: MongoDB

## Installation
1. Klone das Repository:
   ```bash
   git clone https://github.com/deinbenutzername/essens-planer.git
   ```
2. Gehe in das Projektverzeichnis:
   ```bash
   cd essens-planer
   ```
3. Installiere die Abhängigkeiten:
   ```bash
   npm install
   ```
4. Starte die Anwendung:
   ```bash
   npm start
   ```
